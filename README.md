# Rein & Fein Niedersachsen — GitHub Pages Website

Ky paketim përmban një website statik multi-page të optimizuar për GitHub Pages.

## File-at kryesorë
- `index.html` — start page
- `leistungen.html` — faqja e shërbimeve
- `ueber-uns.html` — faqja rreth kompanisë
- `kontakt.html` — faqja e kontaktit
- `impressum.html` — impressum
- `datenschutz.html` — privacy page
- `styles.css` — stilizimi
- `script.js` — menu mobile + reveal animations
- `assets/logo.png` — logo e kompanisë

## Si ta publikosh në GitHub
1. Hape repo-n tënde.
2. Ngarko të gjithë file-at në root.
3. Shko te **Settings > Pages**.
4. Zgjidh:
   - Branch: `main`
   - Folder: `/root`
5. Ruaj.
